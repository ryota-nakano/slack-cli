# 移行ガイド - v1.x から v2.0

## 🔄 何が変わった？

バージョン2.0はモジュラーアーキテクチャによる**完全な書き直し**です。すべてのPythonコードが削除されました。

## ✅ 良いニュース！

**設定変更は不要です！** `.env`ファイルはそのまま使えます。

## 📦 必要なもの

```bash
# 依存関係がインストールされていることを確認するだけ
npm install
```

## 🚀 コマンドの変更

| v1.x コマンド | v2.0 コマンド | ステータス |
|--------------|--------------|---------|
| `python slack_cli.py list` | `node src/index.js channels` | ✅ 同じ機能 |
| `node slack-cli.js channels` | `node src/index.js channels` | ✅ 同じ機能 |
| `node slack-cli.js thread ...` | `node src/index.js thread ...` | ✅ 同じ機能 |

**または npm スクリプトを使用：**
```bash
npm run channels
npm start  # ヘルプを表示
```

## 🔧 環境変数

**変更不要！** 同じ `.env` 形式：

```bash
SLACK_USER_TOKEN=xoxp-...
# または
SLACK_BOT_TOKEN=xoxb-...

# オプション
EDITOR=vim
```

## 🗑️ 削除されたもの

1. **Pythonコード** - すべての `*.py` ファイルを削除
2. **Python依存関係** - `requirements.txt`、`venv/` を削除
3. **旧構造** - モノリシックな `slack-cli.js` を削除
4. **冗長なドキュメント** - 8つのドキュメントを4つに統合

## 🆕 追加されたもの

1. **モジュラー構造** - `src/` に8つの焦点を絞ったモジュール
2. **より良いドキュメント** - README、QUICKSTART、ARCHITECTURE、CHANGELOG
3. **クリーンなコード** - 本番環境対応の品質

**100%機能互換！** 🎉

## 💡 ヒント

### 旧コードにカスタム変更を加えていた場合

新しいモジュラー構造により、カスタマイズが簡単になります：

**旧方式（v1.x）：**
- 770行の `slack-cli.js` を編集
- 何かを壊さないように祈る

**新方式（v2.0）：**
- `src/` 内の特定のモジュールを編集
- 変更が分離されて安全

### Python版を使っていた場合

単純にNode.js版に切り替えるだけ - 同じ機能、同じ使い方！
